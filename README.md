## 📌 Overview

**CausalInferenceML** is a structured research repository for studying and implementing **causal inference** and **advanced time series analysis** in machine learning. It is organized as an end-to-end pipeline — from literature review and dataset exploration, through baseline benchmarking, to advanced model development and final presentation.

The repository is designed to support a reproducible research workflow that anyone can fork, extend, and build upon.

---

## ✨ Key Features

| Module | Description |
|---|---|
| 📚 **Literature Review** | Curated arXiv papers, summaries, and reference notes on causal inference and time series methods |
| 📊 **Dataset Characterization** | Scripts and notebooks for exploratory analysis, preprocessing, and dataset profiling |
| ⚙️ **Baseline Models** | Ready-to-run baseline implementations for fair benchmarking |
| 🧠 **Advanced Models** | Customizable, extensible models for causal inference and forecasting tasks |
| 🎤 **Presentation Materials** | Slides, written reports, and visualizations summarizing findings |

## 🛠️ Installation

**1. Clone the repository**

```bash
git clone https://github.com/Aadip-Thapaliya/CausalInferenceML.git
cd CausalInferenceML
```

**2. Set up a virtual environment** *(recommended)*

```bash
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
```

**3. Install dependencies**

```bash
pip install -r requirements.txt
```
